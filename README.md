# hailong-enterprise
hailong-enterprise là nền tảng quản trị tổng thể của Hải Long, được thiết kế theo kiến trúc module, đóng vai trò làm lõi cho các hệ thống nghiệp vụ như quản lý nhân sự (HRM), quản lý tài sản – máy móc và các hệ thống hiện có. Nền tảng hướng tới chuẩn hóa dữ liệu, thống nhất quy trình, dễ dàng mở rộng và tích hợp
